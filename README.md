# Hypothyroid-Disease-Prediction
This is a project to predict the number of patients who had hypothyroid disease in a clinical camp recently carried out at Nairobi Hospital.

A recent clinical camp at Nairobi Hospital was set up to try and determine whether or not partcicpants suffered from hypothyrodism based on ceratin factors ranging from age, health, sex and certain hormone distribution levels in the body. Th analysis was perfromed using python and took two main approaches namely
